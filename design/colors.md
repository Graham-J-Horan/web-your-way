## Color Hierarchy Guidelines

| Element | Contrast Requirement | Typical Role |
|---------|---------------------|--------------|
| Primary text | 4.5:1 minimum (WCAG AA) | Neutral 900–800 |
| Secondary text | 4.5:1 minimum | Neutral 700–600 |
| Primary buttons | 3:1 for text, 4.5:1 for button text vs background | Primary color |
| Links | 4.5:1 vs surrounding text, plus non-color indicator (underline) | Primary color |
| Disabled states | 3:1 minimum vs background | Neutral 400–300 |
| Focus indicators | 3:1 vs adjacent colors | Primary or accent |

## Note on Usage
- **Primary** should be used for 80–90% of interactive elements
- **Secondary** for 10–20% of supporting actions
- **Accent** for 5% or less — high-impact moments only
- Maintain **3:1 minimum contrast** between any interactive element and its background
